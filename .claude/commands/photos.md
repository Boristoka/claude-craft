## Photo Selection Tips

1. **Neutral colors** - Avoid bright, saturated images
2. **Professional settings** - Real offices, meetings, workspaces
3. **No clichés** - Skip "people pointing at laptops" etc.
4. **Consistent mood** - Same lighting style throughout site
5. **High resolution** - Use `w=1920` for heroes, `w=1200` for sections
6. **Quality parameter** - Always include `q=85` for good compression

---

## URL Parameters

| Parameter | Usage |
|-----------|-------|
| `w=1920` | Hero images (full-width) |
| `w=1200` | Section images |
| `w=400` | Avatars/thumbnails |
| `q=85` | Quality (good balance) |
| `q=80` | Smaller file size |
